# DigitalNightstand

Digital Nightstand for the Tingbot/Raspberry Pi.

Features includes:

* Weather Forecast
* Digital Clock
* Alarm Clock
* Web Frontend

See [DigitalNightstand.tingapp/](DigitalNightstand.tingapp/) for more information.
